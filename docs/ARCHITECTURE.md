## Overview
The Berliz frontend is a single-page application built with Angular and deployed on Netlify.

## Architecture
- Component-based UI using Angular
- Services handle API communication
- Environment-based configuration
- Static asset delivery via CDN

## Key Layers
- Components: UI and presentation logic
- Services: API communication and state handling
- Environments: API endpoints and build configs
- Assets: Images, icons, static files
